# Building with Claude AI

A comprehensive hands-on repository demonstrating Claude API capabilities — from basic chat to advanced features like vision, RAG, tool use, and prompt evaluation.

---

## Project Overview

This project is a structured learning platform built around the Anthropic Claude API. Each module is a runnable, self-contained example that explores a specific Claude capability. Topics covered include multi-turn conversations, streaming, structured outputs, vision, PDF processing, prompt caching, extended thinking, file operations, tool use, RAG (Retrieval-Augmented Generation), and automated prompt evaluation.

---

## Tech Stack

| Layer | Technology |
|---|---|
| LLM API | [Anthropic Claude API](https://docs.anthropic.com) (`claude-haiku-4-5`, `claude-sonnet-4-6`) |
| Embeddings | [Voyage AI](https://www.voyageai.com) (`voyage-3-large`) |
| Language | Python 3.12 |
| Package Manager | [uv](https://github.com/astral-sh/uv) |
| Environment | python-dotenv |

## Prerequisites

- Python 3.12+
- [uv](https://github.com/astral-sh/uv) package manager
- Anthropic API key — [Get one here](https://console.anthropic.com)
- Voyage AI API key (required for RAG modules) — [Get one here](https://www.voyageai.com)

## Configuration

1. Clone the repository:
   ```bash
   git clone https://github.com/chaitanya-jadhav11/building-with-claude-AI.git
   cd building-with-claude-AI
   ```

2. Install dependencies:
   ```bash
   uv sync
   ```

3. Create a `.env` file in the project root:
   ```env
   ANTHROPIC_API_KEY="your-anthropic-api-key"
   VOYAGE_API_KEY="your-voyage-api-key"
   ```

## Running the Project

Each module is independently runnable. Use `uv run` to execute any example:

```bash
# Example: run the interactive math tutor chatbot
uv run python -m app.claude_core.chat_with_claude

# Example: run the RAG pipeline
uv run python -m app.rag.RAG_implementation
```

See the [Project Structure](#project-structure) section below for all available modules and their run commands.

### RAG — Retrieval-Augmented Generation (`app/rag/`)

Build semantic and lexical search pipelines to ground Claude's responses in your data.

| File | Description | Run Command |
|---|---|---|
| `text_chunking_strategies.py` | Three chunking strategies: character-based, sentence-based, section-based | `uv run python -m app.rag.text_chunking_strategies` |
| `text_embeddings.py` | Generate vector embeddings using Voyage AI (`voyage-3-large`) | `uv run python -m app.rag.text_embeddings` |
| `RAG_implementation.py` | Full RAG pipeline: VectorIndex with cosine/euclidean distance, semantic search | `uv run python -m app.rag.RAG_implementation` |
| `BM25_lexical_search.py` | BM25 keyword search index with IDF scoring | `uv run python -m app.rag.BM25_lexical_search` |
| `Multi_Index_search.py` | Hybrid retrieval combining vector and BM25 search strategies | `uv run python -m app.rag.Multi_Index_search` |
